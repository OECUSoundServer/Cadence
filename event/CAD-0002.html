ボカロコンピレーションアルバム - 募集概要 -

ボカロコンピレーションアルバム

主催：ボカロ部

VOCALOIDやUTAU、CeVIO、歌うVOICEROID、NEUTRINO などなどの
合成音声に関する幅広い創作と鑑賞、見聞を広めていくことを目的とする部活です

各個人が、雑談したり、楽曲制作をしたりして、自由に活動しています ( ｀・∀・´)

共催：Cadence

Cadence(旧名：OECUサウンド)は、「学内で音楽を勉強したり、交流したりする機会が少ないことを何とかしたい」という思いで結成されたグループでした。サウンドジャム(短期間でサウンドを制作する)というイベントや、サウンド講習会といった勉強会などのイベントを行いました。最近では外部に向けて、MVを制作したり、二次創作を行ったりもしています。


大阪電気通信大学の音楽系の部とサークルが介して、第2弾の合同イベント開催！！

Cadenceとボカロ部の運営スタッフが作詞・作曲した楽曲を参加者の皆さんがアレンジし、一つのCDにします。

備考 (Notes)

  • 本企画でご制作頂いた楽曲を収録したCDは、有償での頒布を予定しております。
  • 本企画に参加するにあたり、参加費等を含め、費用のお支払いの必要は一切ありません。
  • 参加していただいた方には、本CDを一枚お渡しする予定です。
  • 各作品の著作権は、制作者様ご自身に帰属します。
  • 頒布日以降であれば、個人の作品として Web や物販で自由に公開していただいて結構です。
  • イベントの当落状況により変更される場合があります。
